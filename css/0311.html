Style

head안에 style elem으로 id, class, elem을 설정하여 각각에 맞는 css효과를 줄 수 있음

id>class>elem순으로 우선순위를 갖고 같은 값이면 가장 아래설정의 효과를 가짐

,로 elem 중복설정 가능


여러shtee의 css style을 한번에 변경하고자 할때

css확장자 파일을 새로 만들어 변경할 style을 입력후 변경을 원하는 페이지 head에서 link 입력. (rel은 stylesheet, href는 css확장자 파일명)

internal style sheet와 결합해서도 사용가능.

internal, external, inline으로 모두 css효과를 주지만 lnline이 가장 우선으로 적용되고 IN과 EX는 순서영향없음. 따라서 기존 style우선순위와 동일하게 적용됨

Descendant&child&sibling

descendant 자손 하위의 모든것들. space로 표시

child 자식 바로 밑의 하위개념만. >로 표시

sibling 형제 바로 아랫동생 + 모든동생. ~로 표시

Background

투명도opacity는 elem 요소 전체의 투명도/rgba의 alpha는 background의 투명도(글씨의 투명도는 변하지 않음)

color, image, repeat, positon, attachment등 shorthand로 이용가능

repeat기본적으로 이미지는 반복됨repeat-y 세로 / repeat-x 가로 반복/no-repeat/contain등을 이용하여 조정가능

positiontop,bottom,right,right top등 설정가능하나 해당요소에서의 위치가 아닌 브라우저 전체위치에 위치함

attachmentfixed: 스크롤이 내려가도 배경이 함께 움직이지 않음(기본값 scroll)
