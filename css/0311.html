Style

head안에 style elem으로 id, class, elem을 설정하여 각각에 맞는 css효과를 줄 수 있음

id>class>elem순으로 우선순위를 갖고 같은 값이면 가장 아래설정의 효과를 가짐

,로 elem 중복설정 가능


여러shtee의 css style을 한번에 변경하고자 할때

css확장자 파일을 새로 만들어 변경할 style을 입력후 변경을 원하는 페이지 head에서 link 입력. (rel은 stylesheet, href는 css확장자 파일명)

internal style sheet와 결합해서도 사용가능.

internal, external, inline으로 모두 css효과를 주지만 lnline이 가장 우선으로 적용되고 IN과 EX는 순서영향없음. 따라서 기존 style우선순위와 동일하게 적용됨
